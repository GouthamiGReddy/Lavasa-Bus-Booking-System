# 🚌 Lavasa Bus Booking System

A full-stack Django web application designed to manage university weekend and vacation bus services between Lavasa and Pune.

---

## Features

- Student, Faculty, and Admin roles
- Authentication system
- Weekend bus booking management
- Booking history tracking
- Responsive dashboard
- Admin control panel

---

## Tech Stack

- Django
- Python
- HTML
- CSS
- Bootstrap
- JavaScript
- SQLite

---

## Installation

```bash
git clone <your-repo-link>
cd LavPune-Bus-Booking-System

pip install -r requirements.txt

python manage.py migrate
python manage.py runserver
